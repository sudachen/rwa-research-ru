# Skip Protocol - MEV Infrastructure for Cosmos

## Overview

Skip Protocol is building ecosystem-aligned MEV (Maximal Extractable Value) products on Cosmos. Skip provides cross-chain transaction execution infrastructure and MEV solutions that benefit validators, applications, and users.

## What Skip Builds

### Protocol-Owned Builder (POB)

Skip's POB is a set of Cosmos SDK and ABCI++ primitives:
- Define how apps construct blocks on-chain
- Transparent and enforceable rules
- Protocol-controlled MEV capture
- Customizable block building

### Key Features

1. **On-Chain Block Construction** - Apps control how blocks are built
2. **MEV Recapture** - Protocol captures and redistributes MEV
3. **Transparent Rules** - Enforceable block building logic
4. **Customizable** - Apps define their own rules
