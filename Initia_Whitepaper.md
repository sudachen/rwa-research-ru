# Initia - Interwoven Rollup Ecosystem

## Overview

Initia is a modular blockchain network that combines a Cosmos SDK-based Layer 1 with customizable Layer 2 rollups called Minitias. The platform enables deployment of application-specific chains (Minitias) that operate independently while remaining interoperable through a standardized coordination framework.

## Key Components

### Interwoven Stack

The framework that powers all of Initia's rollups, providing:
- **VM-agnostic architecture** - Support for MoveVM, WasmVM, and EVM
- **Optimistic rollups** - Scalable execution layer
- **Shared security** - Inherited from Layer 1
- **IBC compatibility** - Cosmos interoperability

### Minitias (Layer 2 Rollups)

Application-specific rollups with impressive specifications:
- **High throughput** - 10,000+ TPS
- **Fast finality** - 500ms block times
- **IBC support** - Cosmos ecosystem connectivity
- **Celestia DA** - Data availability integration
- **Multi-VM support** - MoveVM, WasmVM, EVM

### Enshrined Liquidity

Initia's solution to bootstrapping cross-chain liquidity:
- Maintains network security
- Provides native liquidity across rollups
- Eliminates fragmentation

### Vested Interest Program (VIP)

Incentive program fostering long-term ecosystem alignment:
- **VIP Architecture** - Structured incentive system
- **Gauge Voting** - Community-driven allocation
- **VIP Scoring** - Merit-based distribution

### InitiaDEX

Native decentralized exchange for the ecosystem:
- Cross-rollup trading
- Unified liquidity pools
- Seamless asset swaps

### Minitswap

Automated market maker for Minitias:
- Decentralized trading
- Liquidity provision
- Token swaps

## Technical Architecture

### Layer 1 (Orchestration Layer)

- Cosmos SDK-based
- Coordinates security, liquidity, routing, and interoperability
- Manages rollup registration and validation

### Layer 2 (Minitias)

- VM-agnostic (MoveVM, WasmVM, EVM)
- Application-specific optimization
- Independent operation with shared security

## INIT Token
